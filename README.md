# Maya Prompt SaaS

正式 SaaS 版 AI Prompt 管理平台，适合家居产品、AI 场景图、电商广告、品牌摄影 Prompt 的长期沉淀和团队管理。

## 功能

- 现代 SaaS 首页
- 用户登录 / 注册
- Supabase 云数据库
- Prompt 全文搜索
- 文件夹 / 分类 / 标签
- 收藏、热度、版本历史
- 图片链接关联
- CSV / JSON 批量导入
- 管理后台
- AI 优化 / 翻译 / 改写入口
- 一键打开 ChatGPT、Claude、Gemini、Midjourney、Flux、可灵、即梦
- Vercel 一键部署

## 本地运行

```bash
npm install
npm run dev
```
